Layout grid Grid

The grid is based on 12 columns for breakpoints large and up. Smaller breakpoints uses a 2 column layout. The @-sign is used to determine at what breakpoint this grid is used. An element with the classes o-1-2@xsmall-large and o-8-12@large will be 1/2 of the available screen size at breakpoints small to large, and 8/12 of the available size at breakpoints large and up. Hover over an item to see description.

Layout grid.centered Centered grids

Grid items doesn't need to fill the whole available space. You can also center them by adding .u-center instead of .u-left@large.

Example

Centered 6/12
Centered 9/12
Centered 5/12
Centered 2/12

Layout grid.row Row

Rows are used to create a consistent system of margins between elements. Them margin can be different across breakpoints.

Example
This is a regular row. Margin at bottom.
This is a small row. Margin at bottom.
This is a large row. Margin at bottom. 
<div class="o-wrapper kss-wrapper">
  <div class="o-row kss-row">
    This is a regular row. Margin at bottom.
  </div>
  <div class="o-row -small kss-row">
    This is a small row. Margin at bottom.
  </div>
  <div class="o-row -large kss-row">
    This is a large row. Margin at bottom.
  </div>
</div>

Layout grid.row.faux Faux Row

Faux rows are used like rows, but for places that need top padding instead. Useful for first elements in wrappers. Often used together with o-row. Always consider using a row before the element instead of a faux row.

Example
This is a regular faux row. Padding at top.
This is a small faux row. Padding at top.
This is a large faux row. Padding at top. 
<div class="o-wrapper kss-wrapper">
  <div class="o-fauxRow kss-row">
    This is a regular faux row. Padding at top.
  </div>
  <div class="o-fauxRow -small kss-row">
    This is a small faux row. Padding at top.
  </div>
  <div class="o-fauxRow -large kss-row">
    This is a large faux row. Padding at top.
  </div>
</div>
