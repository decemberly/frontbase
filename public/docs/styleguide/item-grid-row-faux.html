Layout grid.row.faux Faux Row

Faux rows are used like rows, but for places that need top padding instead. Useful for first elements in wrappers. Often used together with o-row. Always consider using a row before the element instead of a faux row.

Example
This is a regular faux row. Padding at top.
This is a small faux row. Padding at top.
This is a large faux row. Padding at top. 
<div class="o-wrapper kss-wrapper">
  <div class="o-fauxRow kss-row">
    This is a regular faux row. Padding at top.
  </div>
  <div class="o-fauxRow -small kss-row">
    This is a small faux row. Padding at top.
  </div>
  <div class="o-fauxRow -large kss-row">
    This is a large faux row. Padding at top.
  </div>
</div>
